# CalcPro - Advanced Multi Calculator Suite

A professional web-based calculator toolkit featuring 5 specialized calculators with modern UI/UX design, dark mode support, and full mobile responsiveness.

## ğŸ¯ Project Overview

**Project Type:** Frontend Web Application  

**Problem Statement:**
Users need to rely on different calculators or websites for different types of calculations, which leads to time consumption and an inconsistent user experience.

**Solution:**
CalcPro is a single frontend web application that provides Basic, Scientific, BMI, Age, and Interest calculators on one unified platform with a modern user interface, responsive design, and accurate results.

6. Can switch themes, navigate between calculators, or clear inputs
